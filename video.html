Project 1: MySQL and PHP

This was an individual project that was completed during the spring semester of 2014 for Database Fundamentals(MIS 4300). We were given a set of instructions to guide us in developing a program to interface a MySQL database through a localhost. To test our database and tables, we were given 2 queries to run in our database.

Project 2: EER Diagram

This is an individual project that was completed during the spring semester of 2014 for Database Fundamentals(MIS 4300). We were given a set of business rules and had to draw out an Enhanced Entity Relationship Diagram for a hospital setting. I received an A+ on this project.

Project 3: Java Create and Search Books

This was an individual project that was completed during the spring semester of 2014 for Java Programming (MIS 4310). This is a basic GUI developed using Java Swing to create and search entries for books. The create GUI inputs the data into a CSV file while the search GUI retrieves the data. Save 1 is the java code for search books and Save 2 is a picture of what the application look like when executed.
